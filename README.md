# Hub de links — Fernanda Damasceno de Souza

Página única (`index.html`), sem dependências externas, pronta pra GitHub Pages.
Hub com **duas seções**:
- **Pessoal & acadêmico:** Instagram (@ferdamsceno), LinkedIn, Lattes, ORCID
- **The Deep Sync:** YouTube (@TheDeepSync-PT), Instagram do canal (@thedeepsync.pt)

## ✅ Antes de publicar
1. (Opcional) Trocar as iniciais "FDS" por uma **foto**: coloque `foto.jpg` nesta pasta e siga o comentário no CSS (`.avatar`).

## ⏳ Pendente
- **Dama Evidence:** o cartão está **comentado** no `index.html` (repositório ainda privado).
  Quando publicar, descomente o bloco marcado e troque `SEU-USUARIO` pelo link real.

## 🚀 Publicar no GitHub Pages
1. Crie um repositório **público** no seu GitHub. Duas opções de URL:
   - Nome `links` (ou qualquer nome) → URL fica `https://SEU-USUARIO.github.io/links/`
   - Nome `SEU-USUARIO.github.io` → URL fica `https://SEU-USUARIO.github.io/` (mais limpa)
2. Suba o `index.html` (e a `foto.jpg`, se usar) na raiz do repositório.
3. No repo: **Settings → Pages → Build and deployment → Source: Deploy from a branch → Branch: `main` / `/ (root)` → Save**.
4. Aguarde ~1 min. A URL aparece no topo dessa mesma página de Settings → Pages.

## 🔗 Onde usar a URL
- **Link da bio do Instagram** (o "1 link" — pelo app no celular).
- Pode reaproveitar no LinkedIn (Contato → site) e onde mais quiser.

## ✏️ Como editar depois
É só abrir o `index.html` e mexer no texto/links. Cada bloco `<li>...</li>` é um botão. Para adicionar um link novo, copie um bloco inteiro e troque o `href`, o título (`.t`) e a descrição (`.s`).
